Card News Dashboard

This page will automatically update on the 1st of every month with the latest news and benefit changes across your 18 credit cards. Complete the one-time setup below to activate it.

1 Add your Anthropic API key as a GitHub secret
Repository → Settings → Secrets & variables → Actions → New secret
Name: ANTHROPIC_API_KEY   Value: your API key
2 Enable GitHub Pages
Repository → Settings → Pages → Source: Deploy from a branch
Branch: main   Folder: /docs → Save
3 Trigger your first run manually
Repository → Actions → "Monthly Card News Refresh" → Run workflow
Takes ~2 minutes. This page will be replaced with the live dashboard.
📌 Bookmark your dashboard URL once it's live:
https://<your-username>.github.io/<repo-name>/
The workflow runs automatically on the 1st of each month. You can also re-run it anytime from the Actions tab.
